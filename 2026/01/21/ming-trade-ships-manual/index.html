500-800吨位级远洋贸易帆船 · 结构原理与使用方法

编制单位：元老院造船厂 | 编制时间：1631年 | 密级：内部资料

前言

本手册系统介绍明代及穿越集团建造的东南亚贸易商船——以福船、广船为代表的传统海船，以及H-800型和谐轮系列，涵盖船型结构、帆装系统、航海原理和运营方式。

第一章 概述

明代是中国古代造船业的巅峰时期，专门用于东南亚贸易的商船以福船和广船为代表，载重量通常在500-800吨之间。这些船只承载着”海上丝绸之路”的繁荣，是连接中国与东南亚、南亚乃至中东的重要纽带。

1.1 历史背景

明代中国成为东南亚最大的贸易伙伴，贸易额较高的年份达到白银100万两，相当于东南亚地区当时对外贸易总额。郑和七下西洋后，中国与30多个国家建立了贸易关系，形成了以福建泉州、广东广州为起点的远洋航线网络。

1.2 主要船型

明代专门用于远洋贸易的商船主要包括三大船型：

船型 特点 适用海域
福船 福建建造、尖底深吃水 远洋航行、郑和宝船类型
广船 铁力木制造、结构坚固 战商两用、适合南海
沙船 平底方头、吃水浅 近海运输、载重500-800吨

1.3 500-800吨位商船的地位

载重吨位与用途

  • 500吨级：近海贸易、支线运输、岛际运输
  • 800吨级：远洋贸易主力、主力商船队
  • 1000吨级以上：大型宝船、官方使节船、战船

1.4 《临高启明》中的贸易商船

穿越集团利用现代技术设计建造的H-800型和谐轮系列，借鉴了日本二战期间木质运输船的线型，融入了中式帆装技术，成为500-800吨级贸易船队的核心力量。

第二章 福船——远洋贸易主力船型

福船因在福建沿海建造而得名，是明代远洋贸易和官方使节的主要船型。宋代就有”海舟以福建为上”的评价，郑和下西洋的宝船即为福船船型。

2.1 福船的历史渊源

时期 发展成就
三国时代 吴国在福建沿海设专门造船工场
西晋初期 能建造有8个水密隔舱的海船
唐代 坚固性、稳定性、适航性居世界前列
宋代 福建设造船指挥所，泉州成为世界最大商港
元代 福建大船载重量更胜一筹
明代 福船成为郑和下西洋和抗倭的主力船型

2.2 福船的结构特点

船型特征

  • 船体形状：底尖上阔，首尾高昂，首尖尾方
  • 船底结构：尖底深吃水，稳定性好
  • 船舷：两侧有护板，增强抗风浪能力
  • 长宽比：约2.3-2.4，适合远洋航行

船体尺寸（以郑和宝船为参照）

参数 数值（大型福船）
船长 44丈（约138米）
船宽 18丈（约56米）
载重量 约800吨
船员 数百人至千余人

2.3 水密隔舱技术

福船采用的水密隔舱技术是其最重要的技术创新之一，在泉州宋船和朝鲜新安古船考古中均得到证实。

隔舱数量 功能
前舱 压载、储藏
货舱区 货物存放（2-4个主要货舱）
机舱区 动力设备、锅炉
船员舱 船员住宿、储物
尾舱 操舵设备、船长室

2.4 帆装系统

硬帆结构特点

  • 帆面材料：竹篾、芦苇席或帆布
  • 帆骨结构：硬帆带有撑条，受风效率高
  • 操控方式：升降需要多人配合，但航速快
  • 适应性能：在侧风和顺风时效率尤佳

桅杆配置

桅杆位置 功能
前桅 主帆，航行主力
主桅 最大的帆，提供主要动力
后桅 辅助帆，调整航向
尾桅 方向稳定，有时用于旗号

第三章 广船——坚固耐用的南海主力

广船产于广东，以其坚固的船身和良好的适航性能著称，是明代南海地区最重要的战船和商船船型之一。

3.1 广船的历史与发展

广船由东莞的”乌糟”船和新会的”横江”船演变而来。广州自古以来就是”以舟为车，以楫为马”，是古代海上丝绸之路的重要起点。

3.2 广船的结构特点

材料选择

  • 主要材料：铁力木，比福船的松杉木更坚固
  • 船身结构：下窄上宽，形如两翼
  • 横向结构：紧密的肋骨和隔舱板
  • 纵向强度：依靠龙骨和大胁维持

优缺点分析

优点 缺点
结构坚固耐用 材料昂贵，建造修理费用高
适合装载重型火炮 内海平稳，外洋易晃动
战斗性能优异 损坏后需铁力木修理，难以替代
续航能力强 调遣不如福船便捷

3.3 广船的航海特性

适航区域

  • 最佳海域：里海、内海、近海（海况温和）
  • 次佳海域：南海北部、台湾海峡
  • 受限海域：印度洋深水区、远洋（船身晃动明显）

3.4 广船的战斗配置

装备类型 描述
佛郎机炮 轻型火炮，可连续发射
发熕炮 重型火炮，威力大
火球 用于近战投掷
弓箭 传统武器，用于近距离作战

第四章 沙船——近海运输的中坚力量

沙船是中国古代最主要的船型之一，发源于长江下游，由崇明沙人创用。载重量一般为500-800吨，是明代近海贸易的重要力量。

4.1 沙船的结构特点

船型特征

  • 船体形状：方首方尾，平底，俗称”方艄”
  • 甲板特点：宽敞，型深小，干舷低
  • 船体结构：采用大梁拱，使甲板能迅速排浪
  • 船舷：采用大榔，大中型船每侧有四到六支橹

4.2 沙船的优缺点

优点 缺点
吃水浅，适合浅水航道 平底受水面积大，阻力相对较大
稳定性好，比较安全 排水量不如尖底船，速度稍慢
装货量大，甲板宽敞 不耐风浪，不适合远洋航行
搁浅时易脱困 船头平板，耐浪性差

4.3 沙船的航海性能

帆装与动力

  • 桅帆配置：多桅多帆，一般3-4桅
  • 橹的使用：无风时可用橹推进
  • 航速：理想状态下多帆并用可快速航行
  • 载重能力：500-800吨级为主力

4.4 沙船的主要用途

  • 近海货物运输
  • 长江口及沿海港口间运输
  • 沙石建材运输
  • 客货混装运输
  • 官方物资转运

第五章 H-800型和谐轮——穿越集团的标准化商船

H-800型和谐轮是穿越集团为解决发动机行动运力不足而设计建造的标准化商船，采用流水线批量生产，成为500-800吨级贸易船队的核心力量。

5.1 设计背景

1631年元月，企划院下达第一批24艘H-800标准型建造任务。由施建涛元老主持设计，贯彻模块化理念，做到易改装易修理。最终在香港造船厂的沙滩船台上，提前完成了28艘建造任务。

5.2 技术指标

参数项目 数值
额定载重 800吨
标准排水量 1200吨
满载排水量 1400吨
船型 肥大平底浅吃水型
桅杆数量 3根（标准型）

5.3 设计特点

船型设计

  • 借鉴日本二战期间建造的木质运输船线型
  • 针对17世纪港口条件优化，吃水不宜过深
  • 适合近海运输，最远可达朝鲜半岛、越南、暹罗和日本
  • 搁浅时能被轻易浮起

建造技术

创新技术应用

  • 铁制龙骨和船肋：增强结构强度
  • 助力式手动舵轮：改善操控性
  • 改进型中式帆装：结合现代空气动力学优化
  • 手摇绞车：机械助力设备
  • 铁包木结构：承力较大结构采用铁皮包裹拼接木材

5.4 帆装与动力

类型 描述
动力来源 纯风帆动力，中式帆装
帆装设计 基于20世纪英美中式帆爱好者经验
主要设备 帆布、绞盘、滑车、滑轮组和现代工艺
平均航速 2-4节
理想24小时航行 96海里

5.5 型号变体

型号 特点 代表船只
H-800标准型 标准运载舱，配备船用吊车 EMS号、TNT号、宅急送
H-800a 双桅，加装小型锅炉驱动蒸汽绞盘 -
H-800a1 基于a型改装，前后甲板各装一门火炮 -
H-800s 风帆-蒸汽机混合动力 海丰、海裕、海康、海富
运输难民型 双层铺位，通风系统，载400人 -

5.6 批量生产体系

建造流程

  1. 铁骨、铁肋从博铺加工后运至香港
  2. 在香港建立木材加工厂，加工广东和福建运来的木材
  3. 批量制造标准化木结构部件（桅杆、壳板、甲板条等）
  4. 复杂木结构件由广东当地造船作坊分包加工
  5. 培训工人，提供标准化量具和工具控制公差精度

第六章 船体结构与建造技术

明代及穿越集团的贸易商船采用成熟的木船建造技术，结合水密隔舱、铁骨增强等创新，形成了坚固耐用的船体结构。

6.1 龙骨结构

  • 材料：大型硬木（如铁力木、柚木）
  • 功能：承受船体纵向应力，连接船首和船尾
  • 尺寸：500-800吨级商船龙骨长约20-30米

6.2 船体板材

板材类型 材料 功能
外板 硬木（柚木、铁力木） 直接接触水面，抗腐蚀
内板 杉木、松木 填充结构，增加强度
甲板板 硬木 人员活动，货物堆放

6.3 水密隔舱技术

水密隔舱是中国古代造船技术的重大发明，即使船体部分破损，海水也只会进入少数隔舱，不会导致整船沉没。

隔舱板结构

  • 材料：厚木板，约5-10厘米
  • 连接方式：榫卯结构，铁钉加固
  • 密封方式：桐油石灰混合物填缝
  • 数量：根据船型大小，500-800吨级通常有8-12个隔舱

6.4 铁骨增强技术

  • 应用部位：龙骨、船肋、关键连接点
  • 铁制部件：铁龙骨、铁肋骨、铁箍、铁钉
  • 优势：提高船体强度，延长使用寿命
  • 限制：铁在海水中易腐蚀，需定期维护

6.5 船体连接技术

技术类型 描述
榫卯连接 传统木作工艺，结构牢固
铁钉加固 关键部位使用铁钉连接
铁箍绑扎 桅杆、舵杆等使用铁箍固定
桐油防漏 桐油石灰混合物填缝密封

第七章 帆装系统与航海原理

明代商船的帆装系统是中国古代航海技术的精髓，硬帆结构在中式帆装中具有独特优势，结合多桅多帆设计，能够适应不同风向和海况。

7.1 硬帆结构

硬帆特点

  • 帆面：带有竹篾或木条撑条
  • 材料：帆布、芦苇席或竹篾编织物
  • 受风效率：高于软帆，航速快
  • 操作难度：升降需要多人配合

硬帆优劣势

优势 劣势
受风效率高 重量较大
适合侧风和顺风 升降操作费力
结构简单耐用 逆风航行能力有限
维修方便 需要较多操作人员

7.2 桅杆配置

典型商船桅杆布局：

  • 船首 → 前桅（辅助帆）
  • 前桅 → 主桅（主帆）
  • 主桅 → 后桅（后帆）
  • 后桅 → 尾桅（尾旗杆）

7.3 索具系统

索具类型 功能
牵缩帆索 控制帆面收放
升降索 控制桅杆升降
角索 固定桅杆角度
稳索 保持桅杆稳定
舵绳 控制船舵方向

7.4 航海原理

风向与航向

  • 顺风航行：帆面正对风向，航速最快
  • 侧风航行：帆面与风向成45-90度，航速较快
  • 逆风航行（抢风）：采用”之”字形航线，利用侧风分量推进

航海技术

  • 六分仪：测量天体高度，确定纬度
  • 指南针：确定航向
  • 量天尺：简易天文测量
  • 航海图：记录航线、礁石、港口
  • 牵星术：利用星星确定方位

第八章 运营与使用

明代东南亚贸易商船的运营涉及货物运输、航线管理、船员配置等多个方面，是一项复杂的系统工程。

8.1 货物运输

主要货物类型

出口货物 进口货物
丝绸 香料（胡椒、丁香、肉桂）
瓷器 珍珠、宝石
茶叶 苏木、象牙
铁器、工艺品 木材（檀香木、柚木）
布匹 海产品（燕窝、鱼翅）

8.2 航线网络

明代东南亚贸易主要航线：

  • 中国沿海港口 → 福建泉州 / 广东广州
  • 泉州/广州 → 吕宋/菲律宾
  • 吕宋 → 马尼拉 / 爪哇/印尼
  • 爪哇 → 满剌加/马六甲
  • 马六甲 → 印度古里
  • 古里 → 波斯湾/阿拉伯 / 暹罗/越南

8.3 船员配置

职务 人数 职责
船长 1人 全船指挥，航线决策
大副 1人 协助船长，甲板管理
二副 1人 航海日志，仪器操作
水手长 1人 水手管理，帆缆作业
水手 8-12人 日常操作，值班

木工 1-2人 维修保养

总计 18-24人

8.4 港口操作

进港程序

  1. 接近港口时降帆减速
  2. 利用锚链控制船速
  3. 抛锚停泊
  4. 放下舷梯或搭板
  5. 系泊缆绳
  6. 接受海关和港口检查

装卸作业

  • 利用船用吊车起吊货物
  • 人工搬运装卸
  • 滑轨和滑车辅助
  • 小艇转运散货

8.5 维护保养

定期维护项目

  • 船体清洁：清除海蛎子和藤壶
  • 桐油涂抹：防止木材腐蚀
  • 帆具检查：更换损坏的帆布和绳索
  • 铁件维护：除锈防锈
  • 舱底检查：排除积水，检查渗漏

第九章 综合总结

9.1 技术特点总结

1. 成熟的船型体系
福船、广船、沙船三大船型各有所长，形成了适应不同海域和用途的完整体系。

2. 先进的建造技术
水密隔舱、铁骨增强、榫卯连接等技术的应用，使中国木船在世界居于领先地位。

3. 高效的帆装系统
硬帆结构在中式帆装中具有独特的受风效率，配合多桅多帆设计，适应各种风况。

4. 完善的运营模式
成熟的航线网络、货物贸易体系和船员管理制度，保障了远洋贸易的顺利进行。

9.2 历史意义

  • 推动了海上丝绸之路的繁荣
  • 促进了中国与东南亚的经济文化交流
  • 为后世留下了宝贵的航海技术和经验
  • 奠定了中国在世界航海史上的重要地位

9.3 《临高启明》中的创新

  • 标准化设计：H-800型采用统一图纸，便于批量生产
  • 模块化建造：标准件和流水线作业，提高效率
  • 技术融合：传统工艺与现代技术结合
  • 系列化发展：H-500、H-800、H-1300、H-2000形成系列

第十章 贸易航线与区域地图

明代东南亚贸易航线构成了一个庞大的海上交通网络，连接中国沿海港口与东南亚、南亚乃至中东地区。

10.1 航线总体布局

航线网络从中国沿海出发，经菲律宾群岛、印度尼西亚群岛，穿过马六甲海峡，到达印度古里，最终延伸至波斯湾和阿拉伯地区。

10.2 中国沿海起锚地

福建泉州——海上丝绸之路起点

项目 描述
地理位置 福建省东南部，晋江下游入海口
港口条件 天然深水良港，可停泊大型海船
历史地位 宋元时期世界最大商港，明代仍为重要贸易港口
主要贸易品 丝绸、瓷器、茶叶、铁器、铜钱
造船能力 拥有大型船坞，可建造福船系列

广东广州——南方第一大港

项目 描述
地理位置 广东省珠江三角洲，珠江口西岸
港口条件 河海联运，内河与外海兼顾
历史地位 唐代起成为重要外贸港口，明代设有市舶司
主要贸易品 广纱、广绣、瓷器、漆器
造船能力 广船主要建造基地，技术成熟

10.3 东南亚主要贸易区域

菲律宾群岛区域

  • 吕宋岛：位于菲律宾群岛北部，是中国商船进入东南亚的第一站
  • 马尼拉：吕宋岛西岸港口，连接中国与墨西哥白银贸易的中转站
  • 贸易特点：转口贸易为主，输出中国丝绸瓷器，输入白银、烟草、苏木
  • 航线距离：从泉州到马尼拉约1000海里，航行5-7天

印度尼西亚群岛区域

  • 爪哇岛：东南亚最大岛屿，重要香料贸易中心
  • 苏门答腊岛：扼守马六甲海峡西口，胡椒主产区
  • 婆罗洲：木材、象牙、燕窝产地
  • 贸易特点：香料贸易核心区域，输入胡椒、丁香、肉桂、檀香木
  • 航线距离：从广州到爪哇约2000海里，航行10-14天

10.4 马六甲海峡咽喉要道

满剌加（马六甲）王国

战略地位：满剌加位于马来半岛西南端，扼守马六甲海峡，是东西方海上贸易的必经之地。15世纪成为东南亚最繁荣的贸易港口之一。

港口特征 描述
地理位置 北纬2°，东经102°，马来半岛西海岸
港口条件 天然避风港，水深适宜大型船只停泊
战略价值 东西方航线交汇点，控制印度洋与太平洋连接
贸易货物 香料转运中心，汇集各地商品
航线分叉 向西通往印度，向南通往香料群岛

10.5 南亚印度贸易区域

古里（卡利卡特）——印度西海岸第一大港

  • 地理位置：印度喀拉拉邦北部的科泽科德一带
  • 历史地位：郑和下西洋的重要目的地，明代主要印度贸易港口
  • 主要商品：胡椒、靛蓝、棉布、宝石、珍珠
  • 航线距离：从马六甲到古里约3000海里，航行15-20天
  • 贸易结算：以白银购买印度棉花和香料，运往中国

柯枝（科钦）——香料加工中心

项目 描述
地理位置 印度喀拉拉邦南部，濒临阿拉伯海
港口特色 天然良港，香料加工和贸易中心
主要商品 肉桂、胡椒、豆蔻、椰子制品
与古里关系 互补港口，航线相连

锡兰（斯里兰卡）——印度洋中转站

  • 地理位置：印度半岛南端的岛国，位于阿拉伯海与孟加拉湾之间
  • 战略意义：东西方航线的中继站，船只补充淡水和物资
  • 主要商品：象牙、宝石、椰子、桂皮
  • 历史事件：郑和第二次下西洋时曾在锡兰山立碑纪念

10.6 中东远洋航线

波斯湾区域

航线终点：波斯湾是明代中国商船最西端的贸易区域，主要目的地包括忽鲁谟斯、祖法儿、阿丹等地。

港口 位置 主要商品
忽鲁谟斯 霍尔木兹海峡北岸 宝石、珍珠、香料、波斯地毯
祖法儿 阿曼湾沿岸 乳香、没药、阿拉伯马
阿丹 也门亚丁湾 阿拉伯香料、咖啡、纺织品
天方 麦加附近 伊斯兰圣地进行贸易

航线特点

  • 航行距离：从印度古里到波斯湾约2500海里，航行12-15天
  • 季节规律：利用季风航行，冬季（11-2月）适合西行，夏季（5-9月）适合东返
  • 贸易方式：以物易物为主，中国丝绸瓷器换取波斯宝石和香料

10.7 季节性航线规划

季风航行规律

季节 风向 适合航向 主要航线
冬季（11-2月） 东北季风 南下西行 中国→东南亚→印度
春季（3-5月） 过渡期 灵活安排 区域内贸易
夏季（6-9月） 西南季风 北上东返 印度→东南亚→中国
秋季（10月） 过渡期 灵活安排 返航或避风

典型年度航行周期

500-800吨级商船一年航行计划

  • 11月：利用东北季风从泉州出发，前往马尼拉或爪哇
  • 12月-1月：在东南亚港口装货，补充物资
  • 2月：继续西行至马六甲，等待季风转换
  • 3-4月：前往印度古里，装载香料和棉花
  • 5月：利用西南季风东返
  • 6-7月：返回东南亚，装载当地特产
  • 8月：返回中国，完成年度贸易

10.10 本章小结

航线网络的重要性
明代东南亚贸易航线构成了连接中国与世界的海上通道，马六甲海峡作为咽喉要道，其战略价值延续至今。

区域贸易特色
不同区域有不同的特产和需求，形成了完整的贸易互补体系：中国的丝绸瓷器换取东南亚的香料木材、印度的棉花宝石、中东的珍珠香料。

季风航行智慧
明代航海家充分利用季风规律，选择最佳航行时间，体现了对自然规律的深刻认识和利用。

H-800型的优势
H-800型和谐轮凭借800吨的载重能力和优良的适航性，非常适合执行泉州到波斯湾的全程远洋贸易任务，是穿越集团远洋贸易船队的核心力量。
